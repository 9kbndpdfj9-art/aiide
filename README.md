# AI IDE - Android Application

Android AI IDE - 全功能AI驱动开发环境

## 功能

- **智能代码生成**：基于AI的代码生成
- **多模型路由**：支持Claude、GPT等多种模型
- **技能系统**：可扩展的技能流水线
- **多Agent协作**：DevSwarm多智能体系统
- **语义搜索**：向量搜索和全文搜索
- **代码审查**：自动化代码质量检查
- **零Bug保障**：多文件交叉验证

## 技术栈

- **前端**：Kotlin + Android Views
- **后端**：88个Kotlin引擎
- **搜索引擎**：Rust搜索引擎
- **AI**：Claude API + OpenAI API
